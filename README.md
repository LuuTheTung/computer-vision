Visit this page [Download Python](https://www.python.org/downloads/).
## Install flask
```
pip install Flask
```
## Install flask_cors
```
pip install Flask-Cors
```
## Install numpy
```
pip install numpy
```
## In CMD run yolo_api.py file
```
python yolo_api.py
```
